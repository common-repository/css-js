== Description ==
Attach your custom CSS/JavaScript for single posts or pages

Features

* Attach css/js to single post or page
* Attach css/js from external link (URL)
* Upload Your css/js files
* Manage attached files

Прикрепите пользовательский CSS / JavaScript к одиночной записи или странице

Возможности

* Прикрепление css/js к одиночному посту или странице.
* Прикрепление css/js по внешней ссылке (URL).
* Загрузка css/js из файла.
* Управление существующими файлами.
